# Exploratory Data Analysis - Tourism

This project presents an Exploratory Data Analysis (EDA) on a dataset of famous tourist destinations around the world.

The analysis was developed using Python and Google Colab, focusing on understanding patterns, relationships, and insights related to tourism metrics.

---

##  Objectives

- Analyze tourism data using statistical and visual methods
- Identify patterns and correlations between variables
- Validate hypotheses related to tourism behavior
- Apply data preprocessing techniques for future modeling

---

##  Dataset

The dataset contains information about famous tourist destinations, including:

- Annual visitors
- Entry fees
- Tourism revenue
- Average visit duration
- Location and category of attractions

Data source:  
The dataset was obtained from a public repository on GitHub and used for educational purposes.

---

## Techniques Applied

- Exploratory Data Analysis (EDA)
- Data visualization (histograms, boxplots, heatmaps)
- Correlation analysis
- Data preprocessing
- Normalization (MinMaxScaler)
- Standardization (StandardScaler)
- Dimensionality reduction (PCA)

---

## Key Insights

- Tourism revenue is positively influenced by number of visitors and visit duration
- Data shows asymmetry, with few destinations concentrating high values
- Some variables present moderate correlations, useful for modeling
- PCA helped identify patterns and potential outliers
